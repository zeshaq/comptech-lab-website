Locked baseline for the `dc-lab` disconnected rebuild. The intent is to
mirror OpenShift `4.20.18` plus the operator set used (or planned) in the
previous lab so implementation work does not repeatedly depend on slow
public pulls. Architecture is `amd64` only; the update graph is disabled in
this first install mirror; operator catalog content is pinned to exact
package/channel/version ranges where prior CSV evidence exists rather than
entire channels.

- **OpenShift release payload:** `4.20.18`
- **Architecture:** `amd64`
- **Update graph:** disabled for this first install mirror
- **Active image set:** `plans/disconnected-rebuild/environments/dc-lab/imageset-config.yaml`

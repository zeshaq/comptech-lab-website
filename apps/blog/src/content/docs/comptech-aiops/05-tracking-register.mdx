## ADRs

| ADR | Decision |
| --- | --- |
| `0001 - GitHub Repo As Operational Memory` | Use the `comptech-aiops` GitHub repo as durable project memory. |
| `0002 - Local-First RAG On ZE Tower` | Superseded historical Open WebUI/Ollama RAG experiment. |
| `0003 - ADR-Driven Change Control` | Require issue, milestone, phase, ADR, and changelog linkage for mutating work. |
| `0004 - GitHub Secrets As Secret Custody` | Use GitHub secrets as the project secret custody system until superseded. |
| `0005 - Office MinIO For AIOps Object Storage` | Use office MinIO as the initial object storage backend. |
| `0006 - Custom Portal And Direct Model Serving` | Use a custom portal and direct vLLM/TGI-class serving; exclude Open WebUI and Ollama. |
| `0007 - Go Gin For Custom Portal Backend` | Use Go/Gin for the portal backend/API. |
| `0008 - React Vite TypeScript For Custom Portal Frontend` | Use React, Vite, TypeScript, TanStack, Tailwind, and live updates. |
| `0009 - ze-tower Control Plane And Target Infrastructure Boundary` | Keep `ze-tower` as control plane; target infrastructure remains external. |
| `0010 - Cluster-Resident GitOps Controllers` | Keep Argo CD/OpenShift GitOps inside target clusters. |
| `0011 - Agent Runtime Architecture` | Use LangGraph, Temporal, MCP, OPA, LiteLLM, Postgres, Qdrant, and MinIO. |
| `0012 - Containerized AIOps Control Plane On ze-tower` | Run the AIOps control plane as containers on `ze-tower`. |
| `0013 - Docker Network Boundary Model` | Segment Docker networks by ingress, app, model, observability, and target access. |
| `0014 - Full Phase 1 Tooling Baseline` | Deploy the full Phase 1 control-plane tooling baseline. |
| `0015 - GitHub Actions Deployment Bootstrap To ze-tower` | Use manual GitHub Actions deployment bootstrap to `ze-tower`. |
| `0016 - Public Blog Documentation Mirror` | Keep this blog section as a public-safe mirror of the canonical repo. |
| `0017 - ze-tower Control Plane Backup And Restore Baseline` | Use encrypted Restic backups in MinIO and require a restore drill before production-changing authority. |

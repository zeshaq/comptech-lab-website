## Proposed Policy Shape

Use the existing hub `platform-bootstrap` pattern: ingress-only default deny.

```yaml
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: default-deny-ingress
  namespace: <candidate-namespace>
  labels:
    app.kubernetes.io/name: hub-networkpolicy-baseline
    app.kubernetes.io/part-of: hub-dc-v7-platform
spec:
  podSelector: {}
  policyTypes:
    - Ingress
```

Do not add egress default-deny in the first apply set. The goal is a
low-blast-radius namespace baseline, not a full traffic lockdown.

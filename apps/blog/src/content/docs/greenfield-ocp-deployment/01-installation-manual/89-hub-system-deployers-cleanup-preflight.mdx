## Decision

Do not run a 19-namespace cleanup as the next live change.

Use a single-namespace canary first. The preferred first target is:

```text
platform-bootstrap
```

That namespace currently has no pods or workload controllers.

## Next Gate

Recommended next gate:

```text
OP-GF-COMPLIANCE-19: hub platform-bootstrap system:deployers cleanup canary
```

That gate should remove only:

- `RoleBinding/platform-bootstrap/system:deployers`;
- `ServiceAccount/platform-bootstrap/deployer`.

Then it should wait and confirm whether OpenShift recreates either object
before considering any batch cleanup.
